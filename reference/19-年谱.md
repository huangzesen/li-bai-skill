# 李白年谱（701—762）

| 年份（年龄） | 地点 | 事件 |
|---|---|---|
| 701（1岁） | 碎叶（今吉尔吉斯斯坦） | 生于碎叶城，先世或为陇西李氏，因罪流徙西域。 |
| 705（5岁） | 蜀中绵州昌隆 | 随父入蜀，定居绵州昌隆县（今四川江油），于此启蒙读书。 |
| 710（10岁） | 江油 | 少年好学，博览百家，始习诗赋，兼好剑术纵横之术。 |
| 718（18岁） | 大匡山 | 读书于大匡山（戴天山），从赵蕤学纵横术，隐居数年。 |
| 720（20岁） | 成都 | 谒见益州长史苏颋，受赞"天才英丽"，初露文名。 |
| 724（24岁） | 蜀中→三峡 | 仗剑去国，辞亲远游，出蜀经三峡东下。 |
| 725（25岁） | 扬州、江夏 | 游历扬州，散金三十万，接济落魄之士。至江夏（武昌），登黄鹤楼。 |
| 726（26岁） | 安陆 | 娶前宰相许圉师孙女，入赘许家，定居安陆（今湖北安陆）。 |
| 727—729 | 安陆 | 安家安陆，读书交游，期间屡出游江夏、襄阳等地。 |
| 730（30岁） | 长安、终南山 | 初入长安求仕，谒玉真公主未果，寓居终南山，不得志而归。 |
| 731（31岁） | 安陆 | 归安陆，作《行路难》等诗，抒怀才不遇之慨。 |
| 733—735 | 太原、洛阳 | 客居太原，游洛阳等地，结识豪杰。 |
| 736（36岁） | 山东任城 | 举家迁居山东任城（今济宁），又寓居瑕丘。 |
| 740（40岁） | 山东 | 许氏夫人去世，续纳刘氏，后又与一妇同居。 |
| 742（42岁） | 长安 | 因玉真公主及贺知章荐举，玄宗召入长安，供奉翰林。贺知章见《蜀道难》，呼为"谪仙人"。 |
| 743（43岁） | 长安 | 侍从宫廷，作《清平调》三首、《宫中行乐词》等。沉饮狂放，遭权贵谗毁。 |
| 744（44岁） | 长安→洛阳 | 被"赐金放还"，离开长安。春在洛阳初识杜甫，同游梁宋。 |
| 745（45岁） | 山东 | 与杜甫、高适同游齐鲁，秋杜甫西去，二人此后未再相见。 |
| 746—749 | 江南、齐鲁 | 漫游吴越、齐鲁，纵酒赋诗，与元丹丘等交游。 |
| 750（50岁） | 山东、梁园 | 再游梁宋，续娶宗氏（宗楚客孙女），居梁园。 |
| 751（51岁） | 梁园、江南 | 往来梁园与江南之间，诗名益盛。 |
| 752（52岁） | 幽州 | 北游幽州（今北京一带），察觉安禄山拥兵自重、图谋不轨，忧心忡忡。 |
| 753（53岁） | 宣城 | 南下宣城，游敬亭山、秋浦等地，作《独坐敬亭山》。 |
| 754（54岁） | 宣城、当涂 | 往来宣城、当涂间，与族叔李云作《陪侍御叔华登楼歌》。 |
| 755（55岁） | 宣城 | 安史之乱爆发，天下大乱。白南奔避乱，经溧阳等地。 |
| 756（56岁） | 庐山 | 隐居庐山屏风叠。永王李璘三次遣使聘请，遂入永王幕府。 |
| 757（57岁） | 浔阳 | 永王兵败，白获罪下浔阳狱，多方营救。 |
| 758（58岁） | 浔阳→夜郎 | 判长流夜郎（今贵州桐梓），秋间溯江西行。 |
| 759（59岁） | 白帝城、江夏 | 行至白帝城遇赦放还，作"朝辞白帝彩云名篇"。东归江夏、岳阳。 |
| 760（60岁） | 豫章、浔阳 | 流落江西，依附旧友，贫病交加。 |
| 761（61岁） | 当涂 | 前往当涂投靠县令族叔李阳冰，病重。 |
| 762（62岁） | 当涂 | 十一月，病逝于当涂，葬于龙山东麓。宪宗时迁葬青山。初稿托付李阳冰编《草堂集》。 |
