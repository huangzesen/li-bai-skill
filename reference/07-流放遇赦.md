# 流放遇赦（758—759）

至德二载（757），李白因入永王李璘幕府，坐"附逆"之罪，系浔阳狱中。经御史中丞宋若思及宣慰大使崔涣等人力救，虽免死罪，终被判处长流夜郎。乾元元年（758）春，白自浔阳启程，踏上万里流途。

溯江西行，沿途经江夏、岳阳、洞庭、三峡诸地。所过之处，地方官吏多慕其才名，待之以礼。在江夏，旧友设宴相迎；入洞庭，登楼远眺，感慨万千；至三峡，山高水急，猿声凄绝。白虽身负罪名，然诗兴不减，沿途多有佳作。其流放之诗，既有对冤屈的悲愤，亦有对山川的深情。如《流夜郎题葵叶》中"惭君能卫足，叹我远移根"，寄寓飘零之感；《南流夜郎寄内》中"夜郎天外怨离居，明月楼中音信疏"，写尽思念之苦。

乾元二年（759）春，白行至白帝城一带，忽闻朝廷因关中大旱而颁布大赦令，流罪以下一律赦免。白闻讯狂喜，当即掉转船头，顺流东归。千古名篇《早发白帝城》即作于此际——

> 朝辞白帝彩云间，千里江陵一日还。
> 两岸猿声啼不住，轻舟已过万重山。

全诗一气贯注，轻快飞动。"彩云间"写白帝城之高，"一日还"写水势之速，"啼不住"以声衬速，"万重山"以景化险。此前流途之沉重，此刻尽化为归舟之轻盈。一个"轻"字，既是舟轻，更是心轻——数年冤屈，一朝洗雪，天地重开。此诗被后世誉为李白绝句之冠，亦是中国诗史上最痛快淋漓的归途之歌。

遇赦归后，白沿江东下，经江陵、江夏，复游洞庭、金陵，虽获自由，然经此劫难，其雄心已大不如前，晚年的漂泊与感伤由此而始。
