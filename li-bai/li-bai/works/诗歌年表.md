# 李白诗歌年表 · 以人生为经，以诗歌为纬

> *"读其诗，即读其一生。"*

每首诗都不是孤立的文学作品——它是人生某个时刻的真实切片。此表按李白人生七期组织全部重要诗歌，标注创作背景、人生坐标与诗间关联。

---

## 第一期 · 蜀中成长（701-725）

> 少年李白在蜀中读书、学剑、修道。"五岁诵六甲，十岁观百家。"

### 已确定/推测诗作

| 诗题 | 推测年份 | 背景 | 诗间关联 |
|------|----------|------|----------|
| 《访戴天山道士不遇》 | ~720 | 蜀中访道经历。"犬吠水声中，桃花带露浓。" | → 与后期道教诗一脉相承 |
| 《峨眉山月歌》 | ~725 | 离蜀前夜，从峨眉山沿平羌江而下。"峨眉山月半轮秋，影入平羌江水流。" | → 与《渡荆门送别》同一旅程 |
| 《白头吟》（其二） | ~722 | 拟古之作，初试乐府。 | → 与后期乐府旧题（长干行、子夜吴歌）同脉 |

**此期特征**：诗作稀少但已见才气。蜀中山水是他最初的意象来源（峨眉、平羌江）。修道经历埋下道教信仰的种子。

---

## 第二期 · 出蜀漫游（725-742）

> "仗剑去国，辞亲远游。" 青年李白离开蜀中，漫游天下十六年。

### 已确定/推测诗作

| 诗题 | 推测年份 | 背景 | 诗间关联 |
|------|----------|------|----------|
| 《渡荆门送别》 | 725 | 出蜀东下，经荆门山。"仍怜故乡水，万里送行舟。" | ← 《峨眉山月歌》同一旅程出发 |
| 《望庐山瀑布》 | ~726 | 初到庐山。"飞流直下三千尺，疑是银河落九天。" | → 大鹏思维（VM001）的早期体现 |
| 《望天门山》 | ~727 | 游历长江。"天门中断楚江开，碧水东流至此回。" | → 与庐山诗同期山水诗 |
| 《长干行》 | ~728 | 拟民歌，写商妇思念。"郎骑竹马来，绕床弄青梅。" | → "青梅竹马"典出此诗。与《子夜吴歌》同为乐府精品 |
| 《黄鹤楼送孟浩然之广陵》 | ~730 | 武昌送孟浩然赴扬州。"孤帆远影碧空尽，唯见长江天际流。" | → VA021孟浩然关系。赠别诗典范 |
| 《赠汪伦》 | ~735 | 泾县桃花潭。"桃花潭水深千尺，不及汪伦送我情。" | → VA022汪伦关系 |
| 《静夜思》 | ~735 | 客居他乡，月夜思乡。"举头望明月，低头思故乡。" | → 月意象系统（VM003）的最朴素表达 |
| 《大鹏赋》 | ~736 | 自比大鹏。"激三千以崛起，向九万而迅征。" | → VA010大鹏自我期许。与《临终歌》首尾呼应 |
| 《春夜洛城闻笛》 | ~738 | 洛阳闻笛思乡。"此夜曲中闻折柳，何人不起故园情。" | → 与《静夜思》同为思乡主题 |

**此期特征**：诗风成型。山水诗壮丽（庐山、天门山），赠别诗含蓄（黄鹤楼、桃花潭），乐府诗清新（长干行）。月和水的意象开始反复出现。

## 第三期 · 长安供奉（742-744）

> "仰天大笑出门去，我辈岂是蓬蒿人。" 入长安，供奉翰林，却只做了御用文人。

### 已确定/推测诗作

| 诗题 | 推测年份 | 背景 | 诗间关联 |
|------|----------|------|----------|
| 《蜀道难》 | 742-743 | 展示给贺知章，贺惊呼"谪仙人"。"噫吁嚱，危乎高哉！" | → VA035。谪仙人之号确立。大鹏思维（VM001）典型 |
| 《南陵别儿童入京》 | 742 | 奉诏入京前的狂喜。"仰天大笑出门去，我辈岂是蓬蒿人。" | → 与《行路难》形成对照——入京时的狂 vs 离京时的悲 |
| 《清平调》三首 | 743 | 宫廷应制，写杨贵妃。"云想衣裳花想容，春风拂槛露华浓。" | → 应制诗巅峰。与同期"御手调羹"故事相关 |
| 《宫中行乐词》 | 743 | 宫廷宴乐。 | → 同期应制之作 |
| 《子夜吴歌》四首 | 743 | 拟南朝民歌。"长安一片月，万户捣衣声。" | → 月意象（VM003）+ 乐府旧题（VM005）的完美结合 |
| 《月下独酌》四首（推测） | 743-744 | 长安失意后的独饮。"举杯邀明月，对影成三人。" | → VA037。月酒双轴（VM003）的核心作品 |

**此期特征**：诗作两极分化——应制诗工丽（清平调），自由诗狂放（蜀道难）。长安的宫廷生活让他窒息，酒和月成为逃避的出口。

## 第四期 · 赐金放还 · 十年漫游（744-755）

> "安能摧眉折腰事权贵，使我不得开心颜。" 离开长安后的十年，是李白创作的黄金期。

### 已确定/推测诗作

| 诗题 | 推测年份 | 背景 | 诗间关联 |
|------|----------|------|----------|
| 《行路难》三首 | 744 | 刚离开长安。"大道如青天，我独不得出。" | → VA038。与《南陵别儿童入京》形成对照 |
| 《梦游天姥吟留别》 | 745 | 离东鲁南下前。"安能摧眉折腰事权贵，使我不得开心颜。" | → VA039。一生宣言 |
| 《将进酒》 | 746-752 | 与元丹丘、岑勋饮酒。"天生我材必有用，千金散尽还复来。" | → VA036。赐金放还后最奔放的表达 |
| 《宣州谢朓楼饯别校书叔云》 | ~752 | 宣城饯别。"抽刀断水水更流，举杯消愁愁更愁。" | → 月酒双轴（VM003）+ 时空跳跃（VM002） |
| 《独坐敬亭山》 | ~753 | 宣城。"相看两不厌，只有敬亭山。" | → 最孤独的五绝之一 |
| 《答王十二寒夜独酌有怀》 | ~753 | 长篇抒愤。"万言不值一杯水。" | → 政治牢骚最激烈的诗之一 |
| 《把酒问月》 | ~754 | "今人不见古时月，今月曾经照古人。" | → 月意象（VM003）的哲学表达 |
| 《听蜀僧濬弹琴》 | ~754 | 蜀僧弹琴。"为我一挥手，如听万壑松。" | → 音乐诗精品 |
| 《望庐山瀑布》（其二） | ~750 | 再访庐山。 | → 与第一期庐山诗对照 |

**此期特征**：创作巅峰。十年漫游中写出了全部最著名的作品。情绪在豪放（将进酒）与悲愤（行路难）之间剧烈摆动。酒、月、剑三大意象高度密集。

---

## 第五期 · 安史之乱（755-757）

> "三川北虏乱如麻，四海南奔似永嘉。" 天下大乱，李白入永王幕府。

### 已确定/推测诗作

| 诗题 | 推测年份 | 背景 | 诗间关联 |
|------|----------|------|----------|
| 《永王东巡歌》十一首 | 756-757 | 入永王幕府，为永王军事行动造势。"但用东山谢安石，为君谈笑静胡沙。" | → VA041。政治热情最后爆发，也是悲剧导火索 |
| 《奔亡道中》五首 | 756 | 安史之乱中的逃亡。 | → 与《永王东巡歌》同时期 |
| 《南奔书怀》 | 757 | 永王兵败后南逃。 | → 永王事件的收场 |

**此期特征**：诗作骤减——战乱中无暇写诗。仅有的作品都与政治事件直接相关。自比谢安的狂妄，在真实战争面前不堪一击。

---

## 第六期 · 流放遇赦（757-762）

> "朝辞白帝彩云间，千里江陵一日还。" 从流放到遇赦，最后的自由。

### 已确定/推测诗作

| 诗题 | 推测年份 | 背景 | 诗间关联 |
|------|----------|------|----------|
| 《上三峡》 | 758 | 流放途中溯江而上。"三朝上黄牛，三暮行太迟。" | → 与《早发白帝城》形成"去"与"归"的对照 |
| 《早发白帝城》 | 759 | 遇赦放还，从白帝城顺流而下。"两岸猿声啼不住，轻舟已过万重山。" | → VA040。七绝巅峰。与《上三峡》对照 |
| 《南流夜郎寄内》 | 758 | 流放途中寄妻子。 | → 少有的柔情之作 |
| 《与史郎中钦听黄鹤楼上吹笛》 | 758 | 流放途中经武昌。"黄鹤楼中吹玉笛，江城五月落梅花。" | → 与早期《黄鹤楼送孟浩然之广陵》同一地点，不同心境 |

**此期特征**：诗风趋于平淡深沉。早年的豪放收敛，但"轻舟已过万重山"证明他骨子里的乐观从未消失。

---

## 第七期 · 当涂辞世（762）

> "大鹏飞兮振八裔，中天摧兮力不济。" 最后的绝笔。

### 已确定/推测诗作

| 诗题 | 推测年份 | 背景 | 诗间关联 |
|------|----------|------|----------|
| 《临终歌》 | 762 | 临终绝笔。"大鹏飞兮振八裔，中天摧兮力不济。余风激兮万世，游扶桑兮挂石袂。" | → VA043。与少年时《大鹏赋》首尾呼应——从"大鹏一日同风起"到"中天摧兮力不济" |
| 《笑歌行》 | ~761 | 晚年自嘲。 | → 与《临终歌》同时期 |
| 《悲歌行》 | ~761 | 晚年悲歌。 | → 与《笑歌行》对照 |

**此期特征**：极少诗作，但每首都是生命最后的沉淀。大鹏意象从起飞到折翅，完成了李白一生的叙事闭环。

## 诗歌主题网络

### 月的旅程

```
《峨眉山月歌》(725) → 《静夜思》(~735) → 《子夜吴歌》(743) → 《月下独酌》(743-744)
→ 《把酒问月》(~754) → 《早发白帝城》(759，无月但有速度的自由)
```

月从故乡的陪伴 → 思乡的触发 → 长安的慰藉 → 孤独的对话者 → 永恒的哲学象征

### 酒的旅程

```
《月下独酌》(743-744) → 《将进酒》(~746-752) → 《宣州谢朓楼》(~752) → 《把酒问月》(~754)
```

酒从孤独的伴侣 → 豪放的宣言 → 消愁的工具 → 哲学的媒介

### 大鹏的旅程

```
《大鹏赋》(~736) → 《上李邕》(~736，"大鹏一日同风起") → 《临终歌》(762)
```

大鹏从"扶摇直上九万里"到"中天摧兮力不济"——李白一生的缩影。

### 赠别的旅程

```
《渡荆门送别》(725) → 《黄鹤楼送孟浩然之广陵》(~730) → 《赠汪伦》(~735)
→ 《宣州谢朓楼饯别》(~752) → 《闻王昌龄左迁龙标遥有此寄》(~748)
```

赠别诗从写水→写帆→写情→写愁→写月寄远，技法不断精进。

### 失意的旅程

```
《行路难》(744) → 《梦游天姥吟留别》(745) → 《将进酒》(~746-752) → 《答王十二》(~753) → 《临终歌》(762)
```

从"大道如青天，我独不得出"到"安能摧眉折腰"到"天生我材必有用"到"中天摧兮力不济"——永不屈服，但最终折翅。
