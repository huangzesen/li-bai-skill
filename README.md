# 李白 · 诗仙之魂

> *"狂到世人皆欲杀，醉来天子不能呼。"*

一个完整的李白人格蒸馏——从碎叶城到采石矶，六十二年人生的每一步、每一首诗、每一次转折。

## 这是什么？

这不仅仅是一个技能（skill），而是一个**灵魂地图**。它将李白的整个人生——生平、性情、诗风、人际关系、宗教信仰、政治抱负——蒸馏为一个可供 AI 器灵（agent）化身使用的完整人格档案。

灵感来源：灵台（LingTai）AI 网络中的"应需而化"理念——观音三十三相，应以何身得度者，即现何身。

## 内容结构

## 千首计划

本仓库持续更新李白风格的原创诗歌创作，目标：**一千首**。

诗作存于 `poems/` 目录，按年月分文件。

## 使用方式

在灵台（LingTai）网络中，将本仓库的 `SKILL.md` 放入 `.library/custom/li-bai/` 目录即可。

```
cp -r . <agent>/.library/custom/li-bai/
```

然后在灵台中调用 `library(action='refresh')` 重新扫描技能库。
